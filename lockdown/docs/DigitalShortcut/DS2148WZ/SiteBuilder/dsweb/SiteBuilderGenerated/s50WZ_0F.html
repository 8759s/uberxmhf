DSX50WZ: Spartan-3AN FPGA development board with Ethernet

Xilinx XC3S50AN + WIZnet W5300  =   small, fast, low power web server for only    $119  

Digital Shortcut Inc. DSX50WZ board combines Xilinx's XC3S50AN Spartan-3AN FPGA and WIZnet W5300 Hardwired Newtwork Protocol Ethernet Controller. When your project calls for Ethernet interface it gives you a head start in your embedded project design process. This small factor board may be configured as a very efficient web server and still has most of the XC3S50 FPGA resources available for your application. Just imagine the possibilities of having an Internet Appliance with power consumption less then 1Watt and capable of effectively serving rich web pages with performance on par with "big" servers. Not to mention at the same time having ~33k gates available for your application. The W5300 Ethernet Controller makes complex task of TCP/IP packiet processing and serving web pages no more difficult than working with UART. Hardwired TCP/IP stack handles many of the pakiets without host intervention, 128kB of buffers and paralel interface to FPGA guaraties high performance and most importantly it is easy to comprehend and difficult to hakers. If you are starting a new project with FPGA connected to Ethernet, want to learn FPGA and PicoBlaze programming or just need a web server/network connection for your "older design" it is the perfect board for you.

    Software examples

    This board is shipped preprogrammed with our demo web server software/hardware. Server runs on PicoBlaze microprocessor and is capable of serving selected pages of our web site including this one. When accessed by Chrome browser this very page is served in ~50ms. Web pages in compressed format are stored in internal flash of Spartan3AN. One of the demo pages shows how to access FPGA logic using "XMLHttpRequest" function and java script. Demo comes as a complete working Xilinx ISE WebPACK 10.1 project with all sources and takes less than 33% of FPGA resources. We also included our "SiteBuilder" Tcl/Tk application for converting web pages into a format loadable into FPGA internal flash. Please go to our download page and grab "Software Package" for your board version.
