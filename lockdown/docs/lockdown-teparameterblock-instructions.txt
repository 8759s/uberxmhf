Lockdown Trusted Environment Parameter Block Generation Guide

Author: Amit Vasudevan (amitvasudevan@acm.org)
========================================================================

Note: 
 (a) Operating environment for this writeupwas Ubuntu 10.04
 (b) Repository pathnames are relative to the top-level Lockdown
    project repository


1. Mount trusted environment partition (e.g., /dev/sda3)
   mount /dev/sda3 /mnt/tepart 

2. cd /trunk/code/guestos/windows/genhashes

3. make

4. If all goes well, then you should end up with binary executable
   ./sha1
   
5. ./generate-hashes.pl /mnt/tepart/

6. If everything goes well you will have two binary files
	hashlist_full.dat and hashlist_partial.dat
	
7. cd /trunk/code/guestos/windows/genldnpb

8. /generate-ldnpb.pl hashlist_full.dat hashlist_partial.dat

9. If everything goes well you will end up with binary parameter
   block blob ldnpb_te.bin
