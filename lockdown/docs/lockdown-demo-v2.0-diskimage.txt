Lockdown Demo v2.0 Disk Image

Author: Amit Vasudevan (amitvasudevan@acm.org)
========================================================================

A boot partition with GRUB on /dev/sda1

The untrusted environment is a Windows XP
partition on /dev/sda2
automatic logon with username: guestuser	
	
The trusted environment is a Windows 2003 SP1 (lean configuration)
partition on /dev/sda3
username: Administrator, password: root

The Disk Image contains a Ubuntu 10.04 service partition on
/dev/sda5
username: root, password: root
