Lockdown Hypervisor Build Guide

Author: Amit Vasudevan (amitvasudevan@acm.org)
========================================================================

Note: This is a quick guide. For a detailed explanation 
please see Redmine wiki on 
EMHF: https://plover.pdl.cmu.local/projects/emhf/wiki

1. Checkout lockdown project repository
   svn co https://sparrow.ece.cmu.edu/svn/hyp/lockdown
   
2. Checkout EMHF project repository
   svn co https://sparrow.ece.cmu.edu/svn/hyp/emhf
      
Paths are specified as lockdown/a/b/c or EMHF/a/b/c. Substitute
checked-out working directory for lockdown and EMHF appropriately.

3. cd EMHF/trunk/code

4. ./autogen.sh           				 

5. ./configure --with-approot=<relative path to lockdown/trunk/code> 
      --with-apparchive=emhfapp_lockdown.a --enable-mp=no
      
6. make clean

7. make                   				 

8. If all goes well you end up with binary images init-x86.bin and 
	hypervisor-x86.bin.gz
